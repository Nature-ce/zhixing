## Agent skills

### Issue tracker

Issues 存放在 GitHub Issues，使用 `gh` CLI。external PRs 不进入 triage 队列。参见 `docs/agents/issue-tracker.md`。

### Triage labels

使用默认 label 词汇：needs-triage / needs-info / ready-for-agent / ready-for-human / wontfix。参见 `docs/agents/triage-labels.md`。

### Domain docs

Single-context 布局：根目录下一个 `CONTEXT.md` + `docs/adr/`。参见 `docs/agents/domain.md`。
